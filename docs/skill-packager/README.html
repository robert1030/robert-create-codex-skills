skill-packager 使用說明

這份說明用簡單的方式介紹 skill-packager。目標是讓第一次接觸 Codex skill 的人，也能知道它是什麼、怎麼安裝、怎麼使用。

一句話說明

skill-packager 是一個幫你把 Codex skill 打包成 zip 的小工具。

你可以把它想成「整理書包」：它會檢查 skill 資料夾有沒有放對東西，然後把整包整理成一個可以搬到別台電腦的 zip 檔。

它可以做什麼？

  • 把一個 Codex skill 資料夾打包成 .zip。
  • 打包前先檢查 SKILL.md 是否存在。
  • 確認 skill 名稱和資料夾名稱是否一致。
  • 讓同一個 skill 可以用三種方式安裝。

目前的檔案在哪裡？

檔案或資料夾 用途
skills/skill-packager/ 真正的 Codex skill 資料夾。
skills/skill-packager/SKILL.md 告訴 Codex 什麼時候要使用這個 skill。
skills/skill-packager/scripts/package_skill.py 實際負責打包的 Python 程式。
dist/skill-packager.zip 已經打包好的 zip，可以拿去別台電腦安裝。

三種安裝方式

方式 1：本機個人安裝

如果你只想在自己的電腦使用，就把 skills/skill-packager 複製到 Codex 的 skills 目錄。

Linux、WSL、macOS 可以用：

mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
cp -R skills/skill-packager "${CODEX_HOME:-$HOME/.codex}/skills/skill-packager"

Windows PowerShell 可以用：

$skills = if ($env:CODEX_HOME) {
  Join-Path $env:CODEX_HOME "skills"
} else {
  Join-Path $HOME ".codex\skills"
}

New-Item -ItemType Directory -Force $skills
Copy-Item -Recurse -Force .\skills\skill-packager (Join-Path $skills "skill-packager")

裝好後，重新啟動 Codex。

方式 2：透過 $skill-installer 從 GitHub 安裝

如果你把 skills/skill-packager/ 放到 GitHub repo，別人可以請 Codex 用 $skill-installer 安裝。

$skill-installer install https://github.com/<owner>/<repo>/tree/main/skills/skill-packager

這種方式適合放在 GitHub 上，讓自己或別人從同一個位置安裝。

方式 3：zip 移植安裝

如果你要把它放到隨身碟、雲端硬碟，或傳到另一台電腦，就用 zip。

這個 zip 已經在這裡：

dist/skill-packager.zip

Linux、WSL、macOS 解壓：

mkdir -p "${CODEX_HOME:-$HOME/.codex}/skills"
python3 -m zipfile -e dist/skill-packager.zip "${CODEX_HOME:-$HOME/.codex}/skills"

Windows PowerShell 解壓：

$skills = if ($env:CODEX_HOME) {
  Join-Path $env:CODEX_HOME "skills"
} else {
  Join-Path $HOME ".codex\skills"
}

New-Item -ItemType Directory -Force $skills
Expand-Archive .\dist\skill-packager.zip -DestinationPath $skills -Force

裝好後，應該會看到這種結果：

~/.codex/skills/skill-packager/SKILL.md

Windows 則通常像這樣：

%USERPROFILE%\.codex\skills\skill-packager\SKILL.md

怎麼使用 skill-packager？

如果你有一個 skill 叫 my-skill，放在 skills/my-skill，你可以這樣打包：

python3 skills/skill-packager/scripts/package_skill.py skills/my-skill dist

成功後會得到：

dist/my-skill.zip

zip 裡面應該長這樣：

my-skill/
├── SKILL.md
├── agents/
├── scripts/
├── references/
└── assets/

不是每個 skill 都一定有 agents、scripts、references、assets。有什麼就包什麼。

如果出錯怎麼辦？

錯誤情況 意思 怎麼處理
找不到 SKILL.md 這個資料夾不像標準 skill。 確認你選的是 skill 資料夾，不是上一層資料夾。
資料夾名稱不符合 skill 名稱 SKILL.md 裡的 name 跟資料夾名稱不同。 把兩邊改成一樣。
YAML frontmatter 錯誤 SKILL.md 最上面的設定格式有錯。 檢查最上面的 ---、name、description。

最重要的記法

如果你只記一件事，請記這個：

skill-packager.zip 解壓後，要變成：
Codex skills 目錄 / skill-packager / SKILL.md

只要這個位置對了，Codex 重新啟動後就比較容易讀到這個 skill。
